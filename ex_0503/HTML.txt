웹페이지의 구조를 표현

HTML의 요소
태그로 이루어져 있다.
HTML 요소는 일반적으로 시작태그와 닫는 태그로 이루어져 있다.

HTML의 속성
 - 모든 HTML 요소들은 속성을 가지고 있따.
- 속성은 HTML 요소에 대한 추가적인 정보를 제공한다.
- 특정 태그에서의 속성이나 사전 정의된 속성은 적용된 기능으로서 추가정보를 활용한다.

예시 <img src="경로"> </img>

직접 속성을 만들 수 있다
<div data-sido="인천광역시"></div>

HTML 속성 권장사항

- 소문자로 사용
- 속성은 소문자, 대문자 상관없이 인식하나 소문자로 쓰는 것을 권장
